## Key Insights from Timeline

**What Worked:**
- Starting with ChatGPT prompt (clear goal from day 1)
- Eventually settling on Raspberry Pi (stable, controllable)
- Focusing on ONE platform first (carousel)
- Building infrastructure before scaling

**What Didn't:**
- Cloud detour (learned, but came back to local)
- Trying to do everything at once (now: sequential)

**What's Real:**
- 18 days to get carousel working (not weeks, not months)
- Multiple pivots (PC → cloud → other project → Pi)
- Infrastructure takes time but pays off
- One platform at a time is the way
